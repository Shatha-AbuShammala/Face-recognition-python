# Face Recognition Project

A Python-based face recognition system that identifies individuals from live camera feed. 
This project demonstrates my skills in computer vision, image processing, and algorithmic problem-solving.

## Features
- Detects faces in real-time using a webcam.
- Recognizes registered individuals from a predefined folder (`persons`).
- Draws bounding boxes and labels recognized faces.
- Calculates face distance to improve recognition accuracy.
